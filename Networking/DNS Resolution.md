Record: www.example.com IN A 192.168.0.1
    
    This says 'www.example.com' resolves to the IPv4 address 192.168.0.1. ^5xb8oFBI

AAAA Record: www.example.com IN AAAA 2001:db8::1

    The IPv6 equivalent of record. ^OjCZxN6D

CNAME Record: blog.example.com IN CNAME www.example.com

    This maps 'blog.example.com' to 'www.example.com'. blog.example.com is considered an
    alias for www.example.com ^J4lGCi0e

MX Record: example.com IN MX 10 mail.example.com

    This directs email for 'example.com' to a mail server located at 'mail.example.com'
    The 10 represents priority, lower number, higher priority. ^VYt71bF3

NS Record: example.com IN NS ns1.example.com

    This indicates that 'ns1.example.com' is an authoritative name server for 'example.com'.
     ^YX7ye5jD

PTR Record: 1.0.168.192.in-addr.arpa IN PTR www.example.com

    Used for a reverse lookup using an IP address. this would map '192.168.0.1' back to
    'www.example.com'. You can see this in action by running the command:

    nslookup <your-ip-address-here> ^EW8LZFPZ
